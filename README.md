# API Test to trainee

## Features
* [Json Server](https://github.com/typicode/json-server)

## Routes

**Authentication**: 
* [POST] /auth/token

**Posts**:
* [GET] /posts (list posts)
* [GET] /posts?_page=7&_limit=20 (paginate)
* [GET] /posts/:id (get by id)
* [POST] /posts
* [DELETE] /posts/:id

**Comments**:
* [GET] /comments
* [GET] /comments/:id
* [POST] /comments
* [DELETE] /comments/:id
